# 激光联轴器对准系统

基于 ESP32-S3 和三量三角法激光测距器(19-30 M0)的高精度联轴器对准测量系统。

## 项目结构 

LaserCouplingAlignment_ESP32/
├── hardware/ # 硬件设计文件
├── firmware/ # ESP32 固件代码
│ ├── src/ # 源代码
│ ├── include/ # 头文件
│ ├── lib/ # 第三方库
│ └── platformio.ini # PlatformIO 配置文件
├── software/ # 上位机软件
├── docs/ # 项目文档
├── tests/ # 测试代码
└── README.md # 项目说明文件

## 项目说明

本项目旨在开发一套高精度的联轴器对准测量系统，通过三量三角法激光测距器实现精确的位移测量，并使用 ESP32-S3 进行数据采集和处理。

### 主要特性

- 高精度激光测距
- 实时数据采集和处理
- 无线数据传输
- 可视化对准界面

### 硬件要求

- ESP32-S3 开发板
- 三量三角法激光测距器 19-30（M0）
